
# AI-interactive-app

一个集成了 AI 交互、音乐播放和动态时钟的交互式应用，支持桌面端与移动端适配，提供直观友好的用户体验。

## 项目简介

本项目是一个多功能交互应用，整合了多项实用功能，旨在为用户提供丰富的交互体验：

- **AI 聊天交互**：支持用户与 AI 实时对话，需用户登录后使用，对话内容通过后端 API 与 AI 服务交互（默认可配置为 DeepSeek）。
- **音乐播放器**：具备播放 / 暂停、上一曲 / 下一曲切换功能，支持进度调节，可最小化至悬浮按钮，适配键盘空格键控制。
- **动态时钟**：登录页面展示基于 Canvas 绘制的实时动画时钟，视觉效果流畅。
- **响应式设计**：针对桌面端与移动端设备优化布局，确保不同屏幕尺寸下的使用体验。

## 技术栈

- 前端：HTML、CSS、JavaScript（原生）
- 后端：Node.js、Express
- 开发工具：Visual Studio
- AI 服务：基于 DeepSeek API（可配置其他兼容接口）

## 安装与启动

### 前提条件

- Node.js（v14+ 推荐）
- `npm` 或 `yarn`
- 有效的 AI 服务 API 密钥（如 DeepSeek）
- 数据库环境（MySQL 等，根据配置）

### 步骤

1. 克隆项目到本地

```bash
git clone <项目仓库地址>
cd AI-interactive-app
```

2. 安装依赖

```bash
npm install
```

3. 配置环境变量（在项目根目录创建 `.env`）

示例 `.env`：

```env
# 服务器配置
PORT=3000

# AI 服务配置
AI_API_KEY=sk-你的API密钥
AI_BASE_URL=https://api.deepseek.com/v1
AI_MODEL=deepseek-chat

# 数据库配置
DB_HOST=localhost
DB_USER=你的数据库用户名
DB_PASS=你的数据库密码
DB_NAME=数据库名称
```

> 注意：`.env` 包含敏感信息，已配置忽略提交，请勿将其上传到版本控制。

4. 启动服务

```bash
node server.js
```

在浏览器中访问 `http://localhost:3000` 即可使用应用（如修改了 `PORT`，请访问对应端口）。

## 功能说明

1. AI 聊天交互

- 用户需登录后才能使用聊天功能。
- 通过输入框发送消息，支持回车键提交。
- 后端通过 `/api/chat` 接口转发请求至 AI 服务，返回结果实时展示。
- 自动保留对话历史，清晰区分用户与 AI 消息。

2. 音乐播放器

- 核心控制：播放 / 暂停、上一曲 / 下一曲切换。
- 进度调节：支持拖动滑块调整播放进度，实时更新进度条。
- 界面控制：可最小化至右下角悬浮按钮，点击按钮恢复完整界面。
- 快捷操作：键盘空格键可快速控制播放 / 暂停（输入框未聚焦时）。
- 自动隐藏：操作后一段时间自动隐藏控制栏，优化视觉体验。

3. 动态时钟

- 登录页面展示实时动画时钟。
- 基于 Canvas 绘制，通过 `requestAnimationFrame` 实现流畅动画效果。

## 响应式设计

- 桌面端：完整展示所有功能模块，布局宽松，操作区域充足。
- 移动端：
	- 音乐播放器自适应缩小尺寸（宽度调整为 180px）。
	- 控制按钮位置优化（底部间距调整），适配触控操作。
	- 最小化悬浮按钮位置上移，避开输入区域。

## 注意事项

- 敏感信息保护：`.env` 文件包含 API 密钥、数据库密码等敏感信息，已配置忽略提交，请勿手动上传至版本控制。
- API 密钥：需自行获取有效的 AI 服务 API 密钥并替换，否则 AI 聊天功能无法使用。
- 数据库：确保数据库服务正常运行，且配置与实际环境一致，否则可能影响用户登录等功能。
