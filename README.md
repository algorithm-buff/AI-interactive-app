一个集成了 AI 交互、音乐播放和动态时钟的交互式应用，支持桌面端与移动端适配，提供直观友好的用户体验。
项目简介
本项目是一个多功能交互应用，主要功能包括：
AI 聊天交互：用户可与 AI 进行实时对话
音乐播放器：支持播放控制、进度调节及界面最小化
动态时钟：登录页面展示动画时钟效果
响应式设计：适配桌面端与移动端设备
技术栈
前端：HTML、CSS、JavaScript
后端：Node.js、Express
开发工具：Visual Studio（项目配置可见相关文件）
项目结构
plaintext
AI-interactive-app/
├── .vs/                  # VS配置文件
├── public/               # 前端静态资源
│   ├── assets/           # 资源文件（含时钟动画脚本clock.js）
│   ├── music-player.js   # 音乐播放器逻辑
│   ├── script.js         # 聊天交互核心脚本
│   └── style.css         # 样式文件（含响应式设计）
├── server.js             # 后端服务入口
├── .env                  # 环境变量配置
├── package.json          # 项目依赖配置
└── package-lock.json     # 依赖版本锁定
安装与启动
前提条件
Node.js（v14+ 推荐）
npm 或 yarn
步骤
克隆项目到本地
bash
运行
git clone <项目仓库地址>
cd AI-interactive-app
安装依赖
bash
运行
npm install
配置环境变量创建或编辑 .env 文件，添加必要配置（示例）：
plaintext
PORT=3000
# 其他可能需要的配置（如API密钥等）
启动服务
bash
运行
node server.js
在浏览器中访问 http://localhost:3000 即可使用应用
功能说明
1. AI 聊天交互
通过输入框发送消息与 AI 对话
支持键盘空格键快速发送消息（输入框未聚焦时）
自动展示用户与 AI 的对话历史
2. 音乐播放器
核心功能：播放 / 暂停、上一曲 / 下一曲切换
进度控制：支持拖动滑块调节播放进度
界面控制：可最小化到右下角悬浮按钮，点击恢复完整界面
自动隐藏：操作后一段时间自动隐藏控制栏（提升视觉体验）
3. 动态时钟
登录页面展示实时动画时钟
采用 Canvas 绘制，提供流畅的视觉效果
响应式设计
桌面端：完整展示所有功能模块，布局宽松
移动端：优化界面尺寸与位置，确保触控操作便捷
音乐播放器自适应缩小尺寸
控制按钮位置调整以适配移动端操作习惯
